# MEMORY GAME

Gioco del **Memory** realizzato in **puro JavaScript**, basato su immagini ispirate agli anni ’90 e 2000.

Progetto pensato per esercitarsi con la logica JavaScript e da migliorare progressivamente nel tempo.

---

## Funzionalità

- Griglia di carte coperte
- Scoperta delle coppie uguali
- Completamento del gioco alla scoperta di tutte le coppie

---

## Tecnologie

- HTML
- CSS
- JavaScript (vanilla)

---

## Stato del progetto

Versione base funzionante.
